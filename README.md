# zr

**Release automation for Zig projects. One runner. All targets.**

Status: **pre-alpha / not ready for use**

## What is this?

Zig compiles natively to all targets from a single machine — no cross-compilers or virtual machines needed. `zr` is a release automation tool designed from the ground up to exploit this capability: one CI runner builds binaries for every OS and architecture, then packages them into native formats for each distribution.
